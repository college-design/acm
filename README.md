## 系统技术说明

### 技术第一版V1
- 后台框架:Spring+SpringMVC+Mybatis+Shiro
- 前台模板:bootstrap+anjular+beetl
- 后台:easy-ui

- 后台框架:SpringBoot+Mybatis+Shiro
- 前台模板:bootstrap+anjular+beetl
- 后台:easy-ui
